# Привет! 👋

Я Python-разработчик. Специализируюсь на создании Telegram-ботов, автоматизации рутинных задач и работе с медиа-данными.

### 🛠 Мой технологический стек
- **Язык:** Python (Asyncio, aiogram 3.x)
- **Базы данных:** PostgreSQL, SQLite
- **Инструменты:** Git, Linux (Ubuntu), Docker

### 📂 Мои основные проекты
*   **[DroneLog](ссылка_на_репозиторий_dronelog)** — система логирования и мониторинга данных.
*   **[MediaBot](ссылка_на_репозиторий_mediabot)** — комбайн для обработки аудио и видео контента.
*   **[ConverterBot](https://github.com/0yuman0/converter-bot.git)** — быстрый конвертер файлов различных форматов.
*   **[DownloaderBot](ссылка_на_репозиторий_downloaderbot)** — инструмент для автоматизированного скачивания медиа-контента.
